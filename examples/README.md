# Examples

Runnable screenplays exercising the full screenplay language — frames,
overlays, captions, transitions — end to end.

Run one two ways:

- Paste the file's contents into the Screenplay box in the UI and hit Run.
- POST it as `{"script": "..."}` to `/api/script` (see the README's
  "Directing from the command line").

## Index

- **troll-fable.txt** — a two-character fable about manipulation tactics
  (flattery, urgency, social proof), using frames, lower-thirds, captions,
  and transitions.
